# IPL Sports Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the IPL Matches dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
- IPL Matches Dataset (CSV)

## Analysis Performed
- Dataset information
- Missing values analysis
- Match winners
- Toss winners
- Player of the Match analysis
- Team performance
- Data visualization

## Conclusion
The project analyzes IPL match data and provides insights into winning teams, player performance, and toss statistics using EDA techniques.
